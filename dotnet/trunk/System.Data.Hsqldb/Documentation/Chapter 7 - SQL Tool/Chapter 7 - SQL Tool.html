Chapter 7 - SQL Tool
Chapter Overview

This chapter explains how to use SqlTool.

SqlTool's main function is to read instructions from control files and/or the standard input, executing SQL commands against a database.

The tool has a numerous features related to both interactive and automated use cases.

A short list of features includes:

  • command-line editing
  • embedded procedural language support
  • scripting variables
  • SQL transaction control
  • error handling
